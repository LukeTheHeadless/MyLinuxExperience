## Meu Desktop Enviroment
Eu fui acostumado a customizar mais o **ZorinOS** e o enviroment **XFCE**, mas como minha distro é o **LMDE**(Linux Mint Debian Edition), e vem o **cinnamon** de padrão, eu decidi instalar o XFCE manualmente, o que alguns não recomendam (~~frankendebian, cof cof~~), mas não é tão incomum se pensar que colocam **KDE plasma** em qualquer distro. XFCE é bem leve e tem um modo de customização interessante.

<details><summary>Como instalar XFCE</summary>

Para distros baseadas em Debian/Ubuntu, é simples:
```
sudo apt install xfce4 xfce4-goodies
```

</details>

## Tema
Eu decidi juntar varios temas para meu Desktop.

- **[Ghost Mantis](https://gitlab.com/rlyx/awful-dotfiles/-/tree/master/.themes/Ghost-Mantis)**: deixa as janelas pretas/cinza escuro.
- **[Get-Stickbugged](https://gitlab.com/rlyx/awful-dotfiles/-/tree/master/.themes/Get-Stickbugged)**: deixa as janelas compactas ao nivel de virarem uma linha.
- **[Papirus](https://www.xfce-look.org/p/1166289/)**: Icon pack, que era o padrão no zorinOS?
